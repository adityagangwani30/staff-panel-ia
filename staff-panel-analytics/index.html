Admission Pipeline Definition:
Visualizes progression of active leads through the admission funnel stages.
Formula: COUNT(Leads progressing through Stage Order)
Data Used: Status, Status Order

Stage Conversion

Today's Action Centre Definition:
Critical daily action items for active leads requiring immediate contact.
Formula: Sum of pending, overdue, callbacks, hot and scheduled items
Data Used: Follow-Up Date, Status, Last Call Status

Status Distribution Definition:
Distribution of leads across all CRM statuses.
Formula: COUNT(Leads grouped by Status)
Data Used: Status

Counsellor Performance Definition:
Performance metrics breakdown per assigned counsellor.
Formula: Summarized metrics over Counsellor ID
Data Used: Counsellor ID, Status, Follow-Up Date

Source Performance Definition:
Evaluates leads volume and conversion rates per marketing source.
Formula: (Enrolled Leads per Source ÷ Total Leads per Source) × 100
Data Used: Source, Status

Recent Leads Definition:
List of the 10 most recently added lead records.
Formula: Sort leads by Entry Date descending, take top 10
Data Used: Student Name, Entry Date, Counsellor Name, Status, Source

Staff Performance Analytics Definition:
Detailed performance metrics filtered for the selected staff member.
Formula: Resolve leads based on Staff Selection Hierarchy
Data Used: Counsellor ID, Reports To, Source Centre

Personal Pipeline Definition:
Visualizes personal progression of active leads through stages for the selected staff member.
Formula: COUNT(Leads progressing through Stage Order in staff scope)
Data Used: Status, Status Order, Counsellor ID
Stage Conversion

Personal Status Distribution Definition:
Distribution of statuses for leads assigned within the selected staff scope.
Formula: COUNT(Leads grouped by Status in staff scope)
Data Used: Status, Counsellor ID

Performance Summary Definition:
Summary of performance metrics for the selected staff scope.
Formula: Assigned, Completed (calls > 0), Pending, Overdue, and Enrollment Rate
Data Used: Status, Follow-Up Date, Calls, Counsellor ID
